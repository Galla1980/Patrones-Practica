# Proyecto Creacar - Sistema de Fabricación de Autos

## Consigna del Proyecto

La empresa **Creacar** nos solicita desarrollar un software para la creación de autos. El auto se compondrá de varios atributos: **motor**, **marca**, **modelo** y **cantidad de puertas**. Del motor conocemos el **número** y la **potencia**.

### Requerimientos de Fabricación

Los posibles autos a construir son (todos en sus versiones de **3 y 5 puertas**):

* **Línea Ford**
    * Ford Fiesta (Motor 1.6)
    * Ford Focus (Motor 2.0)
* **Línea Fiat**
    * Fiat Palio (Motor 1.3)
    * Fiat Siena (Motor 1.6)
